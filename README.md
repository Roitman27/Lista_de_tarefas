# 📝 To-Do List - JavaScript Vanilla

Aplicação de lista de tarefas desenvolvida com **HTML, CSS e JavaScript puro (Vanilla JS)**.

O projeto foi criado com o objetivo de reforçar fundamentos de manipulação de DOM, eventos, arrays e persistência de dados utilizando `localStorage`.

---

## 🚀 Funcionalidades

- ✅ Adicionar novas tarefas
- ✅ Marcar tarefas como concluídas
- ✅ Remover tarefas
- ✅ Filtro por:
  - Todas
  - Pendentes
  - Concluídas
- ✅ Contador de tarefas pendentes
- ✅ Limpar tarefas concluídas
- ✅ Persistência de dados com localStorage
- ✅ Suporte a teclado (Enter para adicionar)
- ✅ Layout responsivo para mobile

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

---

## 📱 Responsividade

A aplicação possui layout adaptado para dispositivos móveis utilizando media queries.

---

## 🎯 Objetivo do Projeto

Reforçar conceitos fundamentais de desenvolvimento frontend sem uso de frameworks, focando em:

- Manipulação do DOM
- Gerenciamento de estado
- Eventos
- Estruturação de código
- Responsividade
- Boas práticas de commit

---

2. Abra o arquivo `index.html` no navegador
